## 📖 Sobre o projeto

**Mistérios do Sertão** é um jogo narrativo de escolhas ambientado no nordeste brasileiro. O jogador recebe uma carta antiga do bisavô Sebastião e parte em busca de um tesouro perdido — passando por cenários como o sertão, a mata fechada e uma caverna misteriosa.

Cada decisão influencia o desfecho da história. Existem **dois finais possíveis**: um comum, e um **final secreto** que só se revela para os jogadores mais atentos.

> 💡 **Dica:** o final secreto é desbloqueado por uma combinação específica de escolhas durante a jornada. Será que você consegue encontrá-lo?

## ✨ Funcionalidades

| Recurso | Descrição |
|---------|-----------|
| 🎬 Tela de capa | Apresentação do jogo com título e botão de início |
| 🗺️ 12 capítulos | Jornada completa com múltiplos caminhos |
| 🔒 Botão secreto | Um botão escondido aparece quando o jogador faz as escolhas certas |
| ✨ Efeito hover | Botões ganham vida ao passar o mouse |
| ⬅️ Botões de voltar | Permitem revisitar passos anteriores |
| 🪧 Título dinâmico | A aba do navegador muda conforme o capítulo |
| 🎨 Cenários | Cores e atmosfera diferentes para cada ambiente |
| 🎲 Mensagens aleatórias | Final exibe uma mensagem diferente a cada partida |
| 📓 Diário de bordo | Painel que registra suas descobertas pela jornada |
| 📊 Barra de progresso | Mostra em qual capítulo você está |

## 🚀 Como executar

### Opção 1: Direto no navegador
1. Faça o download dos arquivos do repositório (`Code` → `Download ZIP`)
2. Extraia o arquivo `.zip`
3. Abra o `index.html` com qualquer navegador (Chrome, Firefox, Edge...)

### Opção 2: Clonando o repositório
```bash
git clone https://github.com/JAOG1V1/misterios-do-sertao.git
cd misterios-do-sertao
```
Depois é só abrir o `index.html` no navegador.

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica dos capítulos da aventura
- **CSS3** — visual em estilo cordel, animações e cenários atmosféricos
- **JavaScript (Vanilla)** — controle de fluxo, escolhas, mensagens aleatórias e lógica do final secreto
- **Google Fonts** — tipografia (`Cinzel Decorative` e `Crimson Pro`)

---

## 🎯 Conceitos aplicados

Este projeto coloca em prática diversos conceitos de programação web para iniciantes:

- Manipulação do **DOM** com `querySelector` e `addEventListener`
- Uso de atributos personalizados (`data-proximo`, `data-voltar`, `data-escolha`)
- Trabalho com **classes CSS** para controlar visibilidade e estados
- **Math.random()** para sortear mensagens aleatórias
- **document.title** para alterar o título da aba do navegador
- Lógica condicional (`if`) para desbloqueio do conteúdo secreto
- **Animações CSS** com `@keyframes` e `transition`
- **CSS Variables** para temas e cores consistentes
- Design **responsivo** com media queries
